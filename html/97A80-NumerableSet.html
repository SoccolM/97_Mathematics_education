numerable set


Let X be a set. An enumeration on X is a surjection from the set of natural numbers ℕ to X.

A set X is called numerable if there is a bijective enumeration on X.

It is easy to show that ℤ and ℚ are numerable.

It is a standard fact that ℝ is not numerable. For, if we suppose that the numbers [0,1] were countable, we can arrange them in a list (given by the supposed bijection).

Representing them in a binary form, it is not hard to construct an element in [0,1], which is not in the list.

This contradiction implies that [0,1]⊂ℝ is not numerable.

Remark. If the enumeration ℕ→X is furthermore a computable function, then we say that X is enumerable. There exists numerable sets that are not enumerable.
