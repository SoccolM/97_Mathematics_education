sohcahtoa


Let θ be an acute angle and fix a right triangle having θ as one of its angles. The trigonometric functions sin⁡θ, cos⁡θ, and tan⁡θ can be defined as:

sin⁡θ=oppositehypotenuse
cos⁡θ=adjacenthypotenuse
tan⁡θ=oppositeadjacent

A mnemonic device for remembering the three given above is sohcahtoa.

The sohcahtoa can be divided up into three portions of three letters each. The first portion, soh, should recall to memory that sin⁡θ is opposite over hypotenuse; the second portion, cah, should recall to memory that cos⁡θ is adjacent over hypotenuse; and the third portion, toa, should recall to memory that tan⁡θ is opposite over adjacent.

I would like to thank Mrs. Sue Millikin for teaching me trigonometric functions using this mnemonic device.
