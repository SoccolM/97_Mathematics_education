absolute value inequalities


Let a,b,c∈ℝ and f⁢(x)∈ℝ⁢[x]. There is a mnemonic device that is useful for solving inequalities of the following forms:

a⁢|f⁢(x)|+b≤cc≥a⁢|f⁢(x)|+ba⁢|f⁢(x)|+b<cc>a⁢|f⁢(x)|+ba⁢|f⁢(x)|+b≥cc≤a⁢|f⁢(x)|+ba⁢|f⁢(x)|+b>cc<a⁢|f⁢(x)|+b

Before using the mnemonic device, the expression |f⁢(x)| should be and on the left hand of the inequality. Once this is accomplished, the absolute value must be dealt with: One statement should look to the previous one, the only being that the absolute value are dropped. The other statement should also have the absolute value dropped, but the inequality needs reversed and the number (on the ) needs to be negated.

The two statements as described above should be using either or or and. The mnemonic that aids in remembering which one to use is greator less thand. That is, when the inequality before splitting up has > or ≥, the connector or should be used; when the inequality before splitting up has < or ≤, the connector and should be used.

Here is an example:

8>3+|2⁢x-7|5>|2⁢x-7||2⁢x-7|<5

Since the inequality is <, and should be used.
